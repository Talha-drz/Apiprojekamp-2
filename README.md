# 🍽️ ApiProjeKampi

Modern bir **Restoran Yönetim Sistemi** örneği.  
Bu proje, **ASP.NET Core Web API** ile geliştirilmiş bir backend servisi ve **Razor Pages** tabanlı bir admin panel içermektedir.  
Ayrıca **OpenAI entegrasyonu** ve **SMTP mail sistemi** barındırır.

---

## 🚀 Proje Amacı

Bu proje aşağıdaki senaryoları gerçek bir uygulama üzerinden göstermeyi hedefler:

- RESTful API ile CRUD işlemleri
- API tüketen Razor Pages tabanlı Admin Panel
- OpenAI ile:
  - Otomatik mesaj yanıtı üretimi
  - Tarif oluşturma
  - Menü önerisi üretimi
- SMTP üzerinden şifre kurtarma ve bildirim maili gönderme
- Katmanlı mimari ve DTO kullanımı

---

## 🛠️ Kullanılan Teknolojiler

- .NET 6
- ASP.NET Core Web API
- ASP.NET Core Razor Pages
- Entity Framework Core
- AutoMapper
- Newtonsoft.Json
- OpenAI API (gpt-3.5 / gpt-4)
- SMTP (Gmail)
- Visual Studio 2022

---

## 🧠 OpenAI Entegrasyonları

Projede OpenAI API kullanılarak:

- 📩 Mesajlara otomatik cevap önerisi
- 🍲 Yapay zeka ile tarif oluşturma
- 📋 Menü önerisi üretimi

özellikleri uygulanmıştır.
